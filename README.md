# ⏱️ Pomodoro Timer

Sebuah aplikasi **Pomodoro Timer** sederhana untuk membantu kamu tetap fokus dan produktif.  
Dibuat menggunakan **Next.js + Tailwind CSS**, dengan desain clean dan UX minimalis.

## 🔥 Fitur Utama

- 🎯 **Focus Timer** 25 menit
- 💥 Tombol **Start / Reset**
- 🔄 **Switch ke Break mode** (opsional)
- 📊 Counter **jumlah sesi**
- 🎨 Tampilan responsif dan modern
